•	Structured Database: 
XAMPP and phpMyAdmin are used to create a SQL database that supports a customer billing system. Following the "Pay-As-You-Throw" policy, it determines fees according to the kind and volume of garbage disposal.
•	Semi-Structured Database: 
Customer contracts, which document agreements between the business and its clients, are stored in a NoSQL database that is implemented using Apache Cassandra (Dockerized).
•	Unstructured Database: 
Customer support and complaint handling are handled using an XML database built with BaseX (Dockerized). Additionally, membership access and associated correspondence are tracked by this database.
